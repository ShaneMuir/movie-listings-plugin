Add movie listings to the frontend of your website.

## Description

The Movie Listings Plugin allows you to easily add and manage movie listings on your WordPress site. This plugin is perfect for movie enthusiasts and businesses who want to display movie information in an organized and attractive way.

### Features
- **Custom Post Type**: Create and manage movie listings with a dedicated custom post type.
- **Custom Fields**: Add and update custom fields like movie description, release date, director, stars, and runtime.
- **Shortcodes**: Display movie listings on any page or post using shortcodes.
- **Custom Order**: Reorder movie listings with a custom admin page.
- **Responsive Design**: Ensure your movie listings look great on all devices.

## Installation

1. Upload the `movie-listings` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > Movie Listings to configure the plugin.

## Usage

1. Go to the plugin settings page under Settings > Movie Listings to configure the general settings.
2. Add new movies by navigating to Movies > Add New in the WordPress admin menu.
3. Fill in the movie details using the custom fields provided.
4. Use the `[movies]` shortcode to display movie listings on any page or post.
5. Reorder movies by navigating to Movies > Custom Order in the WordPress admin menu.

## Frequently Asked Questions

### How do I display the movie listings on my site?
You can display movie listings by using the `[movies]` shortcode on any page or post.

### Can I customize the fields for each movie listing?
Yes, you can add details such as movie description, release date, director, stars, and runtime using the custom fields provided in the admin area.

### How do I reorder the movie listings?
Navigate to Movies > Custom Order in the WordPress admin menu to drag and drop movies into the desired order.
